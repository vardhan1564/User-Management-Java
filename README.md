# Java User Management System

A simple Java-based User Management System that allows users to register, log in, and update their profiles.

## Features
- User Registration
- Login Authentication
- Profile Management
- Admin View for Managing Users
- Uses JDBC and MySQL Database

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse

## How to Run
1. Clone this repository
2. Import the project in Eclipse
3. Configure MySQL database connection in `DBConnection.java`
4. Run `Main.java`
